### Почему Подпитка для пламени, а не Элизийский декрет

Если кратко то [Подпитка для пламени](https://www.wowhead.com/ru/spell=391429) из-за баффа. Не обращайте внимания на детали, они вводят в заблуждение.

Если вам срочно не нужно исцеление, не бросайте Глефу в демона, это не стоит ГКД. Он сам довольно быстро умрет от клива к тому же он не наносит вам урон.

[Элизийский декрет](https://www.wowhead.com/ru/spell=390163/) и [Подпитка для пламени](https://www.wowhead.com/ru/spell=391429) наносят первоначальный урон; 
[Элизийский декрет](https://www.wowhead.com/ru/spell=390163/) при использовании способности, [Подпитка для пламени](https://www.wowhead.com/ru/spell=391429) наносит урон, когда демон взрывается. [Подпитка для пламени](https://www.wowhead.com/ru/spell=391429) может проигрывать в цифрах но:

- [Подпитка для пламени](https://www.wowhead.com/ru/spell=391429) также исцеляет на 25% от максимального запаса здоровья, поэтому он по своей сути имеет защитную ценность, помимо увеличения самоисцеления, в то время как [Элизийский декрет](https://www.wowhead.com/ru/spell=390163/) не имеет никакой защитной ценности, кроме генерации 3 душ которые проигрывают 1 душе демона.
- убийство демона, порождает фрагмент души демона. Когда вы поглощаете его (как и любой другой фрагмент), вы получаете +20% урона на 15 секунд. Тем самым в течении 15 секунд все ваши способности наносят на 20% больше урона.
- есть дополнительная защитная ценность, потому что чем больше урона от вы наносите, тем сильнее исцеляетесь!

Хоть может показаться, что [Элизийский декрет](https://www.wowhead.com/ru/spell=390163/) дает больше прибавку, но это не так. Вы можете получить быстрый прирост урона в АоЕ, но если рассматривать нанесенный урон за весь бой либо ключ, [Подпитка для пламени](https://www.wowhead.com/ru/spell=391429) окажется лучше.


## Обзор талантов

### Первый ряд

* [Опустошение Скверной](https://www.wowhead.com/ru/spell=212084) – Фронтальное конусное AoE, которое наносит урон всем целям перед вами и исцеляет, пока вы наносите урон хотя бы одной цели. Наносит хороший АоЕ урон с талантом [Всеобщая тоска](https://www.wowhead.com/ru/spell=390152/) и активирует [Метаморфозу](https://www.wowhead.com/ru/spell=187827) если взят талант [Демонизм](https://www.wowhead.com/ru/spell=213410/)

### Второй ряд

* [Хрупкость](https://www.wowhead.com/ru/spell=389958) – [Печать огня](https://www.wowhead.com/ru/spell=204596/) накладывает дебафф восстанавливающий здоровье в объеме 8% от нанесенного урона. Базовый уровень таланта не такой сильный, но взаимодействие с другими талантами делает ее достаточно мощной. Длится 5 секунд.
* [Огненное клеймо](https://www.wowhead.com/ru/spell=204021) – уменьшает урон, наносимый вам целью, на 40% на 10 сек. Дает защиту только от прямого урона, не работает на войд зоны адов/босса.

### Третий ряд

* [Идеально сбалансированная глефа](https://www.wowhead.com/ru/spell=320387) – Уменьшает время восстановления [Броска боевого клинка](https://www.wowhead.com/ru/spell=204157/).
Раньше это был апгрейд способности 3 ранга. Уменьшает базовое время восстановления до 3 сек., которое затем еще больше сокращается за счет скорости. Не сильный талант, но необходимый для прохождения пути в дереве.
* [Отражающие шипы](https://www.wowhead.com/ru/spell=321028) – [Демонические шипы](https://www.wowhead.com/ru/spell=203720) так же дают 15% парирования на время их действия.
* [Метеоритные удары](https://www.wowhead.com/ru/spell=389724/) – Сокращает время восстановления [Инфернального удара](https://www.wowhead.com/ptr/spell=189110) на 8 сек. Уменьшает время восстановления до 12 секунд. В сочетании с [Беспокойным сердцем Скверны](https://www.wowhead.com/ru/spell=391397) можно сократить 9,6 сек.

### Четвертый ряд

* Ячейка с выбором одного из двух талантов:
  * [Гневное иссечение](https://www.wowhead.com/ru/spell=389997) – увеличивает количество генерируемого гнева с 10 до 20 за одно применение [Иссечения](https://www.wowhead.com/ru/spell=203783). Во время [Метаморфозы](https://wowhead.com/ru/spell=191427) увеличено с 30 до 40.
  * [Разлом](https://www.wowhead.com/ru/spell=263642) – Заменяет способность [Иссечение](https://www.wowhead.com/ru/spell=203783). Способность имеет 2 заряда с коротким временем восстановления, и генерирует больше гнева и душ. Способность наносит больше урона, создает 2 [Фрагмента души](https://www.wowhead.com/ru/spell=219741/) и генерирует 25 гнева, во время [Метаморфозы](https://wowhead.com/ru/spell=191427) способность усиливается и генерирует 3  [Фрагмента души](https://www.wowhead.com/ru/spell=219741/) и 45 гнева.
* [Кальцифицированные шипы](https://www.wowhead.com/ru/spell=389720) – пассивное снижение урона после окончания действия [Демонических шипов](https://www.wowhead.com/ru/spell=203720). Когда время действия [Демонических шипов](https://www.wowhead.com/ru/spell=203720) заканчиваются, автоматически снижается урон на 10%, уменьшающийся на 1% в секунду в течение 10 секунд, пока не достигнет 0.
* Ячейка с выбором одного из двух талантов:
  * [Ревущий огонь](https://www.wowhead.com/ru/spell=391178) – увеличивает исцеление от [Опустошения Скверной](https://www.wowhead.com/ru/spell=212084) до 50% в зависимости от недостающего здоровья. Ранее был кондуитом в Shadowlands. Линейно увеличивает исцеление до 50%, с увеличением на 0% при полном уровне, увеличивая на 1% за каждые 2% отсутствующего здоровья. Это довольно посредственно, поскольку основной лечебный эффект исходит от превращения [Метаморфозы](https://wowhead.com/ru/spell=191427), а не от самого заклинания.
  * [Печать немоты](https://www.wowhead.com/ru/spell=202137) – прерывает чтение заклинаний в области действия. Раньше была базовой способностью. Довольно сильная способность, когда важно прервать группе мобов произнесение заклинаний, или когда вы хотите переместить группу мобов, которые применяют заклинание, потому что прервав заклинание враги побегут к человеку с аггро. Теперь базовая время перезарядки составляет 90 секунд вместо 60.
* [Возмездие](https://www.wowhead.com/ru/spell=389729) – [Демонических шипов](https://www.wowhead.com/ru/spell=203720) отражают получаемый урон. Генерирует в 4 раза больше угрозы. Срабатывает только при фактическом получении урона, а не при уклонении и парировании атак. Становится хуже с улучшением снаряжения.
* [Укрепление пламенем Скверны](https://www.wowhead.com/ru/spell=389705) – Уменьшен магический урон во время действия [Обжигающего жара](https://www.wowhead.com/ru/spell=258920). Ранее был легендарным предметом в Shadowlands и Legion. Уменьшение магического урона на 10%, активно только во время[Обжигающего жара](https://www.wowhead.com/ru/spell=258920). Довольно посредственный талант.

### Пятый ряд

* [Взрывная душа](https://www.wowhead.com/ru/spell=247454) – эта способность наносить урон по области, для которой в качестве ресурса используются [Фрагменты души](https://www.wowhead.com/ru/spell=219741/), а также гнев. Способность поглощает все доступные души для нанесения урона, урон зависит от количества потребленных душ. Используемые души все равно будут вас исцелять. Сведите к минимуму свое движение, чтобы не уменьшить случайные поглощения душ. Также накладывает [Хрупкость](https://www.wowhead.com/ru/spell=389958) на все цели которые заденет способность. Способность нельзя использовать если нет душ, даже если у вас достаточно гнева.
* [Пиршество душ](https://www.wowhead.com/ru/spell=207697) – дает слабый ХоТ после каждого [Раскалывания душ](https://www.wowhead.com/ru/spell=228477). ХоТ обновляется, а не складывается, так что это не очень хорошо.
* [Пламенная боль](https://www.wowhead.com/ru/spell=207548) – увеличивает продолжительность [Обжигающего жара](https://www.wowhead.com/ru/spell=258920) и увеличивает скорость передвижения во время его действия. Увеличивает продолжительность от 1,5/3 секунды до 7,5/9 секунд и скорость передвижения на 10/20%. Имеет неплохую синергию с другими талантами, но сама по себе способность слабая.
* [Удлиненные шипы](https://www.wowhead.com/ru/spell=389721) – увеличивает продолжительность [Демонических шипов](https://www.wowhead.com/ru/spell=203720). Продолжительность увеличивается от 1/2 секунды до 7/8 секунд. Все сопутствующие эффекты от других талантов также будут работать в этот период. Увеличивает эффективное время работы способности примерно на 17/33%.
* [Пылающая кровь](https://www.wowhead.com/ru/spell=390213) – пассивно увеличивает наносимый урон от огня. Увеличивает на 5/10% наносимого урона от огня. Может дать большую прибавку урона, особенно если использовать таланты, наносящие урон от огня.
* Ячейка с выбором одного из двух талантов:
  * [Призрачный барьер](https://www.wowhead.com/ru/spell=263648) – дает нам способность, которая создает щит, размер щита увеличивается увеличивается за каждую поглощенную душу. В настоящее время в большинстве случаев талант не будет иметь места для применения, но его можно будет использовать в качестве дополнительной защиты.
  * [Массовое извлечение](https://www.wowhead.com/ru/spell=320341) – дает нам способность, которая наносит небольшое количество урона всему поблизости, а затем немедленно создает и поглощает 5 душ, что в сумме составляет 30% от урона, полученного за последние 5 секунд, минимум 5% от максимального хп. Талан посредственный, но дает небольшую прибавку урона.
* [Печать цепей](https://www.wowhead.com/ru/spell=202138) – это талант, позволяющий собирать мобов в одной точке. Перезарядка 60 секунд. Это притягивает врагов к центру печати через 2 секунды после ее размещения, прерывая большинство применений способностей и замедляя их. Он невероятно мощен для контроля паков мобов и сбивания некоторых заклинаний, которые обычно нельзя прервать. Он также замедляет врагов на 70% на 6 секунд, что позволяет вам кайтить после его использования. Некоторые более крупные элитные враги могут быть невосприимчивы к эффекту способности.

### Шестой ряд

* [Страж Бездны](https://www.wowhead.com/ru/spell=268175) – улучшает [Хрупкость](https://www.wowhead.com/ru/spell=389958), теперь она дополнительно снижает урон по вам, и [Раскалывание душ](https://www.wowhead.com/ru/spell=228477) тоже будет накладывать эффект [Хрупкости](https://www.wowhead.com/ru/spell=389958).
* [Жаркий выброс](https://www.wowhead.com/ru/spell=227174) – [Обжигающий жар](https://www.wowhead.com/ru/spell=258920) генерирует души при первом попадании. Предполагаемый шанс создание души с КАЖДОЙ цели составляет 60-70%. С помощью этой способности вы можете создать максимум 5 душ. Если у вас более 5 душ, все лишние души, генерируемые этим талантом, автоматически расходуются, что значительно повышает выживаемость.
* [Разрушительный бастион](https://www.wowhead.com/ru/spell=326853) – [Опустошение Скверной](https://www.wowhead.com/ru/spell=212084) восполняет больше здоровья а 100% исцеления превращается в щит на 10 сек.
* [Бурлящая огненная кровь](https://www.wowhead.com/ru/spell=390808) – когда [Обжигающий жар](https://www.wowhead.com/ru/spell=258920) наносит критический урон вы накапливаете дополнительный гнев. Срабатывает только один раз за тик, включая первый удар. Он имеет хорошую синергию с критическим ударом и генерирует довольно много гнева.
* [Наслаждение болью](https://www.wowhead.com/ru/spell=343014) – Дает щит после окончания времени действия [Огненного клейма](https://www.wowhead.com/ru/spell=204021). 

### Седьмой ряд

* [Кузня души](https://www.wowhead.com/ru/spell=391165) – пассивный бафф к урону после накопления 10 душ. Ранее был кондуитом в Shadowlands. Дает 40% бонус к урону [Раскалывания души](https://www.wowhead.com/ru/spell=228477) или [Взрывная душа](https://www.wowhead.com/ru/spell=247454) после поглощения 10 душ. Талант срабатывает от способностей, которые поглощают души.
* [Вестник боли](https://www.wowhead.com/ru/spell=207387) – пассивное снижение урона после поглощения душ. Ранее была талантом артефакта в Легионе. Дает пассивное уменьшение урона после каждого поглощения души, суммируется. В среднем можно поддерживать ~3-4 стака, иногда до 10.
* [Дар созерцателя тьмы](https://www.wowhead.com/ru/spell=389708) – сокращает время восстановления [Опустошения Скверной](https://www.wowhead.com/ru/spell=212084) и восстанавливает немного гнева после применения способности. Раньше это был легендарным предметом в Shadowlands, но с немного с другим способом срабатывания. Работает только в том случае, если [Опустошения Скверной](https://www.wowhead.com/ru/spell=212084) было применено полностью. Кроме того, восстановление гнева и сокращение времени восстановления рассчитываются отдельно. Может сократить время восстановления до 36–48 секунд (48–56 с 1 вкинутым талантом) и восстановить 20–40 гнева (10–20 с 1 вкинутым талантом).
* [Огненное погребение](https://www.wowhead.com/ru/spell=389220) – увеличен урон от огня по целям, пораженным [Огненным клейма](https://www.wowhead.com/ru/spell=204021).
Ранее был талантом артефакта в Легиона. Это позволяет нам наносить на 40 % больше урона от огня любой цели, на которую действует [Огненное клеймо](https://www.wowhead.com/ru/spell=204021), в том числе и на те на которое оно распространилось. Превращает его из оборонительной способности в ДПС кулдаун.
* [Печать гнева](https://www.wowhead.com/ru/spell=389715) – Увеличена область действия всех печатей. Увеличение области действия на 2 метра означает увеличение площади примерно на 44%, так как оно увеличивает радиус всех печатей с 10 метров до 12. Однако размер анимации не увеличивается, поэтому сложнее определить зону поражения.

### Восьмой ряд

* [Сосредоточенное раскалывание](https://www.wowhead.com/ru/spell=343207) – увеличен урон [Раскалывания души](https://www.wowhead.com/ru/spell=228477) по цели в таргете. Этот талант увеличивает ваш урон СТ урон на 30%. Он также работает в AoE, но только против вашей основной цели. Это делает [Раскалывания душ](https://www.wowhead.com/ru/spell=228477) более конкурентоспособным в СТ.
* [Собиратель душ](https://www.wowhead.com/ru/spell=389711) – избыточное исцеление от [Фрагмента душ](https://www.wowhead.com/ru/spell=219741/) дает вам щит. Раньше это была азеритовый талант в Battle for Azeroth, но с небольшими изменениями. 
* [Поддержание пламени](https://www.wowhead.com/ru/spell=393827) – увеличивает урон от [Опустошения Скверной](https://www.wowhead.com/ru/spell=212084). Просто пассивное увеличение урона на 40% [Опустошения Скверной](https://www.wowhead.com/ru/spell=212084). Ничего особенного в этом нет.
* [Сожжение заживо](https://www.wowhead.com/ru/spell=207739) – распространяет [Огненное клеймо](https://www.wowhead.com/ru/spell=204021) на соседние цели. ДоТ распространяется каждые 2 секунды на ближайшего врага, на котором еще нет ДоТа. На всех врагов на которых распространился ДоТ будет действовать эффект уменьшения наносимого урона и увеличение получаемого урона.
* [Цикл запечатывания](https://www.wowhead.com/ru/spell=389718) – сокращает время восстановления всех печатей на 3 секунды, когда одна из них применяется и срабатывает. Раньше это был азеритовый талант в Battle for Azeroth. Как и в случае с [Печати души](https://www.wowhead.com/ru/spell=395446), применяется только один раз за каждую печать из-за внутреннего времени восстановления. Печать должна поразить цель, чтобы талант сработал. Сокращение перезарядки действует и на применяемую печать, так как эффект восстановления  срабатывает после срабатывания печати.

### Девятый ряд

* [Уязвимость](https://www.wowhead.com/ru/spell=389976) – враги на которых действует [Хрупкость](https://www.wowhead.com/ru/spell=389958) получают больше урона. Дает 4/8% дополнительного урона целям, пораженным [Хрупкостью](https://www.wowhead.com/ru/spell=389958). Хороший прирост урона, а также увеличивает исцеление.
* [Кормление демона](https://www.wowhead.com/ru/spell=218612) – уменьшает время восстановления [Демонических шипы](https://www.wowhead.com/ru/spell=203720) при поглощении [Фрагмента душ](https://www.wowhead.com/ru/spell=219741/). Сокращает время восстановления на 0,25/0,5 сек. за каждую поглощенную душу. В среднем сокращает время восстановления на 20+ сек. в минуту, увеличивающееся с ускорением. Существенное увеличение времени работы [Демонических шипы](https://www.wowhead.com/ru/spell=203720).
* [Обугленная плоть](https://www.wowhead.com/ru/spell=336639) – [Обжигающий жар](https://www.wowhead.com/ru/spell=258920) продлевает действие [Огненного клейма](https://www.wowhead.com/ru/spell=204021) каждый раз, когда наносит урон. Увеличивает продолжительность на 0,25/0,5 сек. за каждый тик, включая первый удар. Эффект продления так же срабатывает на распространенное клеймо талантом [Сожжение заживо](https://www.wowhead.com/ru/spell=207739).

### Десятый ряд

* [Сокрушение души](https://www.wowhead.com/ru/spell=389985) – позволяет стакать [Хрупкость](https://www.wowhead.com/ru/spell=389958). Это очень сильный талант. Все остальные эффекты от талантов в дереве также применяются, то есть мы получаем 8% самоисцеления, 4% снижения урона и 8% дополнительного урона за стак. Кроме того, мы получаем 2 стака на нашу основную цель при применении с помощью [Раскалывания души](https://www.wowhead.com/ru/spell=228477) или [Взрывной души](https://www.wowhead.com/ru/spell=247454), что делает его чрезвычайно мощным. В среднем мы можем ожидать от 2 до 5 активных стеков (4-10 на основной цели).
* [Разрубатель душ](https://www.wowhead.com/ru/spell=207407/) – наносит большой урон от огня и генерирует души с перезарядкой 1 мин. Ранее был способностью артефакта а Легионе. На самом деле он наносит половину AP урона оружия в левой руке, накладывает на цель огненный ДоТ и генерирует души. Он довольно мощный, особенно в сочетании с другими талантами, так как он обеспечивает генерацию душ и сам по себе наносит хороший урон.
* [Последнее прибежище](https://www.wowhead.com/ru/spell=209258/) – вы не умрете если получите смертельный урон (чит дес). Перезарядка 8 минут, после срабатывания активирует [Метаморфозу](https://wowhead.com/ru/spell=191427). Один из самых слабых чит десов из существующих, но он всегда можно взять в талантах когда это требуется. Талант может уберечь от вайпа группу или рейд и сохранить боевое воскрешение. Вы можете использовать его, чтобы избежать механика, если в этом есть потребность. Раз в 8 минут вместо того, чтобы получать смертельный урон, вы восстанавливаете здоровье до 30% и активируете [Метаморфозу](https://wowhead.com/ru/spell=191427) с ее полной продолжительностью. 
* Ячейка с выбором одного из двух талантов:
  * [Подпитка для пламени](https://www.wowhead.com/ru/spell=391429) – призывает демона, которого можем убить только мы, после смерти демон взрывается и роняет [Душа демона](https://www.wowhead.com/ru/spell=163073). Ранее была способностью ковенанта Некролорды в Shadowlands. Случайным образом вызывать демона на 25 секунд, который получает дополнительный урон от [Броска боевого клинка](https://www.wowhead.com/ru/spell=204157/) в среднем частота появления демона 1,58 раз/мин, после смерти наносит урон по площади максимум 5 целям, цели после 5 получают уменьшенный урон. 
  * [Элизийский декрет](https://www.wowhead.com/ru/spell=390163) – печать, которая наносит большое АоЕ урон и генерирует несколько душ. Ранее была способностью ковенанта Кирии в Shadowlands. Хороший бурст урон, генерирует 3 дополнительные души за одно применение.

Сокращение перезарядки на 15 секунд, поэтому каждый заряд способности перезаряжается за 45 секунд. Возможно будет имеет некоторое нишевое применение для максимизации ДПС.

    var $tabs = function (target) {
      var
        _elemTabs = (typeof target === 'string' ? document.querySelector(target) : target),
        _eventTabsShow,
        _showTab = function (tabsLinkTarget) {
          var tabsPaneTarget, tabsLinkActive, tabsPaneShow;
          tabsPaneTarget = document.querySelector(tabsLinkTarget.getAttribute('href'));
          tabsLinkActive = tabsLinkTarget.parentElement.querySelector('.tabs__link_active');
          tabsPaneShow = tabsPaneTarget.parentElement.querySelector('.tabs__pane_show');
          // если следующая вкладка равна активной, то завершаем работу
          if (tabsLinkTarget === tabsLinkActive) {
            return;
          }
          // удаляем классы у текущих активных элементов
          if (tabsLinkActive !== null) {
            tabsLinkActive.classList.remove('tabs__link_active');
          }
          if (tabsPaneShow !== null) {
            tabsPaneShow.classList.remove('tabs__pane_show');
          }
          // добавляем классы к элементам (в завимости от выбранной вкладки)
          tabsLinkTarget.classList.add('tabs__link_active');
          tabsPaneTarget.classList.add('tabs__pane_show');
          document.dispatchEvent(_eventTabsShow);
        },
        _switchTabTo = function (tabsLinkIndex) {
          var tabsLinks = _elemTabs.querySelectorAll('.tabs__link');
          if (tabsLinks.length > 0) {
            if (tabsLinkIndex > tabsLinks.length) {
              tabsLinkIndex = tabsLinks.length;
            } else if (tabsLinkIndex < 1) {
              tabsLinkIndex = 1;
            }
            _showTab(tabsLinks[tabsLinkIndex - 1]);
          }
        };

      _eventTabsShow = new CustomEvent('tab.show', { detail: _elemTabs });

      _elemTabs.addEventListener('click', function (e) {
        var tabsLinkTarget = e.target;
        // завершаем выполнение функции, если кликнули не по ссылке
        if (!tabsLinkTarget.classList.contains('tabs__link')) {
          return;
        }
        // отменяем стандартное действие
        e.preventDefault();
        _showTab(tabsLinkTarget);
      });

      return {
        showTab: function (target) {
          _showTab(target);
        },
        switchTabTo: function (index) {
          _switchTabTo(index);
        }
      }

    };

    var mytabs = $tabs('.tabs');
    if (localStorage.getItem('mytabs')) {
      mytabs.showTab(document.querySelector('[href="' + localStorage.getItem('mytabs') + '"]'));
    }

    document.addEventListener('tab.show', function (e) {
      localStorage.setItem('mytabs', e.detail.querySelector('.tabs__link_active').getAttribute('href'));
    })
